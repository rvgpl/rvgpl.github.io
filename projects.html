Zapnotes

Real time notes collaboration

A realtime notes collaboration platform, multiple persons can collaboarate and edit notes in realtime. No need to signup, just share the draft link and get working. Works on all modern devices.

Built using bootstrap, node.js, socket.io.

Check it out at www.zapnotes.in

Minimal Blog

A responsive blog theme for Tumblr

Designed with mobile first approach,responsive and looks good on any device. Single Column layout for mobile and two column layout for any other larger viewport.

Added support for changing header background, user avatar, description and accent colors. Theme uses Disqus for comments.
